## Overview 
This repository contains phishing URL investigation reports completed as part of my SOC Analyst training and self-learning. Each investigation follows a structured incident analysis methodology to identify malicious indicators, assess threats, and document findings in a professional report format.

The objective of these projects is to demonstrate practical skills expected from a Security Operations Center (SOC) Level 1 Analyst, including phishing analysis, threat intelligence, IOC identification, and technical report writing.

## Skills Demonstrated
-  Phishing URL Analysis
-  Threat Intelligence Investigation
-  Domain & WHOIS Analysis
-  DNS Record Analysis
-  URL Reputation Assessment
-  IOC Extraction & Documentation
-  MITRE ATT&CK Mapping
-  Technical Report Writing
